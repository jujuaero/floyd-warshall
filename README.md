UTILISATION DU PROGRAMME

1. Lancer interface.py
2. Entrer le numero du graphe a etudier
3. Cliquer sur "Charger graphe"
4. Cliquer sur "Executer Floyd-Warshall"
5. Si besoin, saisir un sommet source et un sommet destination pour afficher un chemin minimal


- Fichier graphe: graphes/graphe15.txt
- Trace d'execution: trace_execution/graphe15_trace.txt
- Description de la source et de la methode de construction: source_graphe15.md

Le graphe 15 est une modélisation du reseau metro parisien (lignes 1, 4 et 14),
avec stations reelles et arcs bidirectionnels entre stations consecutives.
